```typescript
interface CreationForm {
  world: string;              // 世界ID
  worldConfig: WorldConfig;   // 世界详细配置

  gender: string;             // 性别
  age: string;                // 年龄阶段

  talents: string[];          // 天赋列表
  legacy: LegacyItem[];       // 继承遗产
}

interface WorldConfig {
  name: string;
  era: string;
  description: string;
  features: string[];
}

interface LegacyItem {
  name: string;
  type: string;
  effect: string;
  icon: string;
}
```

**状态**: 类型完整，无 `any`，通过 TypeScript 严格模式检查

## 端到端流程验证

完整的用户流程现已打通：

```
注册/登录 → Dashboard主页 → 新建人生 → CreationPage
                                              ↓
                                    5步骤创建角色
                                              ↓
                                    调用 saveApi.create()
                                              ↓
                                    保存到 PostgreSQL
                                              ↓
                                    自动跳转 /game
                                              ↓
                                    GamePage 加载存档
                                              ↓
                                    显示角色信息+操作按钮
                                              ↓
                                    返回 Dashboard
                                              ↓
                                    点击"继续此生"
                                              ↓
                                    重新进入 /game
```
